# DAILY PROTOCOL

## Every Morning

1. Open BADAWY-1.
2. Read Today's Mission.
3. Review Today's Tasks.
4. Start the Next Action.
5. Ignore new ideas until today's work is finished.

---

## During The Day

- Record every new idea.
- Record every important decision.
- Record every solved problem.

---

## Before Sleeping

- Mark completed tasks.
- Write what I learned today.
- Decide the first task for tomorrow.

---

## Golden Rule

Never end the day without knowing the first task of tomorrow.
